【游戏名称】连连看

【游戏介绍】
这是一款H5原生开发的益智类小游戏。游戏中，你需要将相同图案的图案连接起来消除，直到所有图案都被消除才能过关。

【玩法说明】
操作：
- 点击相同的两个图案，图案之间有不超过两个转折点的连接路径
- 点击"提示"按钮，游戏会显示一对可以连接的图案
- 点击"洗牌"按钮，图案会随机重新排列
- 点击"重新开始"按钮，游戏重新开始

规则：
- 图案之间必须可以通过不超过两个转折点的路径连接
- 只能连接相同图案的两个元素
- 需要消除所有图案才能过关

目标：
- 完成所有关卡
- 每一关都有时间限制，需要在时间内完成
- 图案种类会随着关卡增加而增加

Tips：
- 使用提示功能可以帮助你在卡住时继续游戏
- 先寻找明显连通的路径，减少后期难度
- 洗牌功能虽然会打乱图案，但也可能创造新的连接机会

【技术介绍】
HTML5原生开发，主要语言是JavaScript，使用Hilo.js游戏框架开发，并添加了移动端优化代码。
